## Метод Ньютона (`newton/`)

### Описание

**Метод Ньютона** (метод касательных) — итерационный метод нахождения корня уравнения `f(x) = 0`. На каждом шаге строится касательная к графику функции в текущей точке, а следующее приближение выбирается как точка пересечения касательной с осью абсцисс.

**Итерационная формула:**

```
x_{n+1} = x_n - f(x_n) / f'(x_n)
```

Итерации продолжаются до тех пор, пока `|x_{n+1} - x_n| < tol`.

Если аналитическая производная не передана, она вычисляется **численно** методом центральных разностей:

```
f'(x) ≈ (f(x + h) - f(x - h)) / (2h)
```

### Функция `newton()`

```python
from newton.newton import newton

newton(f, x0, df=None, tol=1e-7, max_iter=1000, diff_step=1e-7)
```

**Параметры:**

| Параметр    | Тип        | По умолчанию | Описание |
|-------------|------------|-------------|----------|
| `f`         | `callable` | —           | Функция, корень которой ищем (`f(x) = 0`) |
| `x0`        | `float`    | —           | Начальное приближение |
| `df`        | `callable` | `None`      | Производная `f`. Если не передана — вычисляется численно |
| `tol`       | `float`    | `1e-7`      | Допустимая погрешность: итерации останавливаются при `|x_{n+1} - x_n| < tol` |
| `max_iter`  | `int`      | `1000`      | Максимальное число итераций |
| `diff_step` | `float`    | `1e-7`      | Шаг `h` для численного дифференцирования (используется только если `df=None`) |

**Возвращает:** `float` — найденное приближение корня.

**Исключения:**

- `ValueError` — если производная в текущей точке близка к нулю (метод неприменим).
- `ValueError` — если метод не сошёлся за `max_iter` итераций.

### Примеры использования метода Ньютона

**Пример 1 — корень `f(x) = x² - 2` (аналитическая производная):**

```python
import math
from newton.newton import newton

f  = lambda x: x**2 - 2
df = lambda x: 2 * x

root = newton(f, x0=1.0, df=df)
print(f"x = {root:.10f}")  # x = 1.4142135624  (≈ √2)
```

**Пример 2 — корень `f(x) = cos(x) - x` (производная вычисляется численно):**

```python
import math
from newton.newton import newton

f = lambda x: math.cos(x) - x

root = newton(f, x0=0.5)  # df не передаётся
print(f"x = {root:.10f}")  # x ≈ 0.7390851332
```

**Пример 3 — корень `f(x) = x³ - x - 2`:**

```python
from newton.newton import newton

f  = lambda x: x**3 - x - 2
df = lambda x: 3 * x**2 - 1

root = newton(f, x0=1.5, df=df, tol=1e-10)
print(f"x = {root:.10f}")  # x ≈ 1.5213797068
```

Готовые примеры находятся в файле [`newton/example.py`](newton/example.py).

## Метод простых итераций (`iteration_method/`)

### Описание

**Метод простых итераций** (метод последовательных приближений) решает уравнение `f(x) = 0`, предварительно преобразовав его к виду `x = φ(x)`. Начиная с начального приближения `x₀`, строится последовательность:

```
x_{n+1} = φ(x_n)
```

**Условие сходимости:** метод сходится, если в окрестности корня выполняется `|φ'(x)| < 1`.

Итерации останавливаются при достижении заданной точности: `|x_{n+1} - x_n| < epsilon`.

### Функция `simple_iteration()`

```python
from iteration_method import simple_iteration

simple_iteration(phi, x0, epsilon=1e-6, max_iter=10000)
```

**Параметры:**

| Параметр    | Тип        | По умолчанию | Описание |
|-------------|------------|-------------|----------|
| `phi`       | `callable` | —           | Итерационная функция `φ(x)` (уравнение приведено к виду `x = φ(x)`) |
| `x0`        | `float`    | —           | Начальное приближение |
| `epsilon`   | `float`    | `1e-6`      | Требуемая точность: итерации останавливаются при `|x_new - x_old| < epsilon` |
| `max_iter`  | `int`      | `10000`     | Максимальное число итераций |

**Возвращает:** `float` — найденное приближение корня.

**Исключения:**

- `ValueError` — если метод не сошёлся за `max_iter` итераций. Необходимо проверить начальное приближение или условие сходимости `|φ'(x)| < 1`.

### Примеры использования метода простых итераций

**Задача:** найти корень уравнения `x³ - x - 2 = 0`.

Приведём к виду `x = φ(x)`: возьмём `φ(x) = (x + 2)^(1/3)`.

```python
from iteration_method import simple_iteration

def phi(x):
    return (x + 2) ** (1 / 3)

root = simple_iteration(phi, x0=1.0, epsilon=1e-8)
print(f"x ≈ {root:.8f}")          # x ≈ 1.52137971
print(f"Проверка: {root**3 - root - 2:.2e}")  # ≈ 0.00e+00
```

Готовые примеры находятся в файле [`iteration_method/example.py`](iteration_method/example.py).

## Вспомогательный скрипт `ditohomiya.py`

Экспериментальный скрипт в процессе разработки. Читает математическое выражение из командной строки, токенизирует его (разбивает на числа и операторы: `+`, `-`, `*`, `/`, `//`, `%`, `^`, `sqrt`, `sin`, `cos`, `exp`, `log`) и вычисляет результат с помощью `eval()`.

Поддерживаемые математические операторы:

| Оператор | Действие                  |
|----------|---------------------------|
| `+`      | Сложение                  |
| `-`      | Вычитание                 |
| `*`      | Умножение                 |
| `/`      | Деление                   |
| `//`     | Целочисленное деление     |
| `%`      | Остаток от деления        |
| `^`      | Возведение в степень      |
| `sqrt`   | Квадратный корень         |
| `sin`    | Синус                     |
| `cos`    | Косинус                   |
| `exp`    | Экспонента                |
| `log`    | Натуральный логарифм      |

> ⚠️ Скрипт находится в стадии разработки. Функция дихотомии (`dichotomy()`) является заглушкой.
